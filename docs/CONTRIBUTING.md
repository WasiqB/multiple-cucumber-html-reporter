# Contributing

We try to achieve a 100% coverage on the Javascript code. Please make sure that each addition reaches this goal.

## Commit Message

Please make sure to follow the [commit message](https://github.com/conventional-changelog/commitlint#what-is-commitlint) format.

## Pre-requisites

- Node.js
- pnpm

## Setup

Install dependencies:

```shell
pnpm install
```

## Tests

- clone the project to your local machine
- do a `pnpm install`

To create a report use `pnpm test`.

To create a report and also check the coverage use `pnpm unit.test.coverage`.

To open the generated report use `pnpm report`.
